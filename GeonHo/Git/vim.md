## 모드

- Normal (편집을 주로 하는 모드)
  - 들어가는 방법: esc
- Insert (실제로 글을 작성하는 모드)
  - 들어가는 방법: i, a, o, I, A, O
- Visual (드래그)
  - 들어가는 방법: v (Normal 모드에서)

## 이동키
숫자 + 이동키
### 문자 단위
  - j (아래)
  - k (위)
  - h (왼쪽)
  - l (오른쪽)
### 단어 단위
  - w, e (앞으로)
  - b, B (뒤로)
### 문장 단위(newline, \n)
  - $ (문장 끝으로)
  - 0 (문장 앞으로)
  - ^ (문장 앞으로, 공백이 아닌 곳으로)
### 파일 단위
  - gg (파일 앞으로)
  - G (문장 끝으로)

## 복사, 붙여넣기
y: yank

연속으로 2번 <- 해당 줄에 대해서 작업을 한다.
yy: 지금 있는 줄을 복사해라.
dd: 지금 있는 줄을 지워라.
p: paste

삭제해도 버퍼에 들어간다. ex) d d p
