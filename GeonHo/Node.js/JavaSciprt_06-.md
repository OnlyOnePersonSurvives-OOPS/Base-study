## 7. 함수

### 함수의 정의

~~~
function f123(){
    console.log(1);
    console.log(2);
    console.log(3);
}

f123();
console.log(`-----------`);
f123();
console.log(`-----------`);
f123();
~~~

### 함수의 입력과 출력

입력
~~~
console.log(Math.round(1.6));
console.log(Math.round(1.4));

function sum(first, second){ // parameter
    console.log(first + second);
}

sum(2, 4); // argument
~~~
